# HUFSEATS 🍜

한국외대~경희대 근처 맛집을 기록하고, 룰렛으로 메뉴를 랜덤 추천받는 모바일 전용 웹 애플리케이션.

## 🚀 기술 스택

- **Frontend**: React, Vite, JavaScript
- **Styling**: TailwindCSS
- **Deployment**: Vercel

---

## 📱 주요 기능

- 🔁 오늘 뭐 먹지? 메뉴 룰렛
- 🍽️ 사용자 등록 맛집 리스트
- 🗺️ 지도 기반 전체 맛집 보기 (미구현)
- 📝 후기 작성 및 열람 (미구현)
